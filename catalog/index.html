Modules

USB/MIDI Master Module

Ref : MM102

Master Module to connect your keyboard modules to a MIDI compatible device (Mac, PC, iPhone, iPad...)

Provides 2 MIDI assignable knobs (volume, effect control..) and 4 MIDI assignable buttons (octave-shift, program change, triggers..)

This module is shipped with a dedicated configuration application used to control various settings such as keyboard velocity curves, MIDI channel output...

 

Specifications

  • Material : Aluminium (black finish) for the enclosure
  • USB-B connector for power and MIDI connectivity for Mac and PC (cable included)
  • Low power consumption : can be powered by an iPhone or and iPad using the appropriate USB-to-Lightning cable (not included)
  • 6.35 mm (1⁄4 in) TS (Jack) socket for connecting a sustain pedal (not included)
  • Weight : 500g (1.1 lbs)
  • Enclosure dimensions : W:6 L:18 H:5 cm (2.4x7x2 in)

24-Keys Module

Ref : KM101

24 full-size keys to add two more octaves to your Piano de Voyage.

Specifications

Keys : Spring action keybed (i.e. not weighted) with 24 velocity-sensitive, full-size, square-front plastic keys

Material : Powder-coated aluminum (black) for the enclosure.

Weight : 1kg (2.2 lbs)

Dimensions : 33x18x5 cm (13x7x2 in)

25-Keys Module (end side)

Ref : KM102

Add 25 keys at the right-end of your Piano de Voyage

Specifications

Keys : Spring action keybed (i.e. not weighted) with 24 velocity-sensitive, full-size, square-front plastic keys

Material : Aluminium (black finish) for the enclosure.

Weight : 1kg (2.2 lbs)

Dimensions : 33x18x5 cm (13x7x2 in)
